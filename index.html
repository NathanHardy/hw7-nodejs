Mister Web Developer person. I would like for you to tell me about what is the Node.

Node is essentially a set of libraries written on top of Google's V8 JavaScript engine for Chrome. These libraries help to simplify networking and certain server-side functionality. Wikipedia defines Node.js as follows:

Node.js is an open source cross-platform runtime environment for server-side and networking applications. Node.js applications are written in JavaScript...

Similar functionality can be achieved with more traditional server-side programming languages, such as PHP, however there are differences. For example, Node includes an API for non-blocking I/O, which means that multiple connections can be handled on a single thread, each of which can idle, but cannot sleep, which would delay execution of other connections. This makes Node ideal for realtime web applications. Another nice feature is a library built into Node which allows applications to serve as a web server without requiring Apache HTTP or a similar solution. These and other features have made Node a fast growing and highly in-demand technology, which is now used by GoDaddy, Groupon, LinkedIn, Microsoft, PayPal, SAP, Walmart, and Yahoo, among many others.
